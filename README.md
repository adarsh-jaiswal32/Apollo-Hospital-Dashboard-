# 📌 Project Overview

The **Apollo Hospitals Leads Flow Dashboard** is an interactive healthcare analytics dashboard developed using Microsoft Power BI to monitor patient flow, hospital occupancy, diagnosis trends, doctor feedback, and billing performance.

This dashboard helps healthcare management teams analyze operational efficiency, patient distribution, and revenue insights through real-time visual analytics and KPI tracking.

The project demonstrates practical business intelligence, healthcare analytics, dashboard storytelling, and data visualization skills.

---

# 🚀 Dashboard Features

## ✅ Key Functionalities

* Interactive Patient Filtering
* Dynamic Date Range Selection
* Bed Occupancy Analysis
* Doctor Feedback Distribution
* Diagnosis-Wise Patient Analysis
* Billing Amount Monitoring
* Healthcare KPI Tracking
* Patient Flow Analytics

# 📈 Dashboard Components

## 1️⃣ Bed Occupancy Analysis

This section visualizes occupancy across different hospital bed categories:

* Private Beds
* General Beds
* ICU Beds

### Key Insights:

* Private beds show the highest occupancy.
* ICU occupancy remains comparatively lower.
* Bed allocation trends help optimize hospital resource planning.

---

## 2️⃣ Feedback Volume per Doctor

A donut chart visualizes patient feedback distribution among doctors.

### Business Value:

* Helps evaluate doctor performance.
* Supports service quality improvement.
* Identifies patient engagement patterns.

---

## 3️⃣ Diagnosis-Wise Patient Count

This section analyzes the number of patients across major diseases:

* Viral Infection
* Flu
* Malaria
* Typhoid
* Pneumonia

### Key Insights:

* Viral infections recorded the highest patient count.
* Pneumonia showed the lowest diagnosis volume.
* Seasonal disease trends can assist healthcare planning.

---

## 4️⃣ Billing & Patient Trend Analysis

A trend analysis comparing:

* Health Insurance Amount
* Patient Diagnosis Count

### Insights:

* Billing amount fluctuates significantly across treatment periods.
* Patient diagnosis volume remains relatively stable.
* Revenue spikes indicate periods of high patient activity.
